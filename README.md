# Krishna's Portfolio 🚀

Welcome to my developer portfolio website! This is a modern, responsive portfolio showcasing my projects and skills.

## ✨ Features

- **Modern Design** - Clean and professional GitHub-inspired dark theme
- **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- **GitHub Integration** - Automatically fetches and displays your latest projects
- **Smooth Animations** - Elegant scroll animations and transitions
- **Customizable** - Easy to personalize with your own information

## 📁 Project Structure

```
├── index.html      # Main portfolio page
├── styles.css      # Modern styling with dark theme
├── script.js       # GitHub API integration & interactivity
└── README.md       # This file
```

## 🎯 Sections

1. **Hero Section** - Welcome message with call-to-action buttons
2. **About Me** - Personal introduction
3. **Featured Projects** - Automatically loads your 6 latest GitHub repositories
4. **Skills & Technologies** - Showcase your technical skills
5. **Contact** - Get in touch section with links

## 🚀 Getting Started

1. This portfolio is hosted on GitHub Pages
2. All your GitHub repositories are automatically displayed in the Projects section
3. The website updates automatically as you add new repositories

## 🎨 Customization

Edit these sections in `index.html`:

- **Hero Section**: Change your name and subtitle
- **About Me**: Update your bio and description
- **Skills**: Add/remove technologies you work with
- **Contact**: Add more social media links

### Change Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #0d1117;
    --secondary-color: #161b22;
    --accent-color: #1f6feb;
    --text-color: #c9d1d9;
    /* ... more colors ... */
}
```
